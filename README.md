# stack_PTIT

### 1. Clone repo

```bash
git clone https://github.com/m4l0n6/stack_PTIT.git
cd stack_PTIT
```

### 2. Cài đặt các dependencies
```bash
npm install
```

### 3. Thêm env
```
CHECK_TIMEOUT = 10 // kiểm tra interval Tailwind CSS

PORT = 3000
```

### 4. Chạy web 
```bash
npm run dev
```
